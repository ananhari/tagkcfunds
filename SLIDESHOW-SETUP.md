# TAG KC Slideshow Setup Guide

## Quick Start

1. **Add your images** to the `images/` folder (any name, any supported format)
2. **Run the index generator** (choose one method below)
3. **Open** `fundraiser-with-slideshow.html` in your browser
4. **Done!** Your slideshow will display all images automatically

---

## Supported Image Formats

- JPG/JPEG
- PNG
- GIF
- WEBP
- BMP
- SVG

---

## Running the Index Generator

Choose the method that works best for you:

### Option 1: Python (Recommended)
```bash
python3 generate-images-index.py
```

### Option 2: Bash (Linux/Mac)
```bash
./generate-images-index.sh
```

### Option 3: Node.js
```bash
node generate-images-index.js
```

### Option 4: Windows
Double-click `generate-images-index.bat` or run:
```cmd
generate-images-index.bat
```

---

## Adding New Images

Whenever you add new photos to the `images/` folder:

1. Just copy the new images into the folder
2. Run the index generator again
3. Refresh your browser

That's it! The script automatically detects all images regardless of their names.

---

## How It Works

The slideshow loads images from the `images/images.json` file, which is automatically generated by the scripts. The JSON file contains:

```json
{
  "images": [
    "photo1.jpg",
    "vacation-pic.png",
    "IMG_1234.jpeg",
    "any-name-you-want.jpg"
  ],
  "count": 4,
  "folder": "images",
  "generated": "auto"
}
```

---

## Slideshow Features

- **Auto-play**: Automatically cycles through images every 5 seconds
- **Manual controls**: Use Previous/Next buttons
- **Pause/Play**: Toggle auto-play
- **Indicators**: Click dots to jump to specific images
- **Responsive**: Works on mobile and desktop
- **Smooth transitions**: Fade effects between slides

## Customization

Edit these values in `fundraiser-with-slideshow.html`:

```javascript
const SLIDE_INTERVAL = 5000;  // Time between slides (milliseconds)
const IMAGE_FOLDER = 'images/';  // Folder containing images
```

## Troubleshooting

### Images not showing?
1. Check that the `images/` folder exists
2. Make sure image files are in supported formats
3. Run the index generator script
4. Check browser console (F12) for errors
5. Ensure `images/images.json` was created

### "Permission denied" error?
Make scripts executable:
```bash
chmod +x generate-images-index.py generate-images-index.sh generate-images-index.js
```

### Need to reset?
Delete `images/images.json` and run the generator again.
